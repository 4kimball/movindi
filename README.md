### 컴포넌트 구성

- App.vue

  - header(컴포넌트 x)
    - header의 메뉴를 누르면 router로 페이지 이동
  - Main
    - 영화목록 (포스터만 보여주고)
      - 영화 상세페이지 -> 캐러셀
    - 개인 추천 영화
    - 오늘의 배우 -> 일단 랜덤
    - 커뮤니티 페이지 간략히 보여주기
  - **서비스 소개 페이지**
    - 무빈디 소개하기
  - **우리 배우들**
    - 배우검색기능
    - 배우목록 (무한 스크롤)

  - **커뮤니티**
    - 영화리뷰 - 댓글
    - 구인구직
    - 자유게시판[옵션]
  - 프로필
    - 팔로우 팔로워
    - sns ID (인스타, 카카오톡) -> 연락처
    - 프로필정보
    - 필모 -> 클릭하면 영화로 이동
    - 내가 좋아요한 영화
    - 내가 작성한 리뷰
  - 검색결과 페이지 -> 검색했을 때 새로운 페이지로 이동
    - 검색결과가 없습니다 -> 아래의 영화는 어떠세요
    - 검색 성공했을 때 -> 해당 영화 출력 -> 해당 영화와 비슷한 영화 추천

- 로그인-회원가입-로그아웃

  - 새로운 페이지로 이동
